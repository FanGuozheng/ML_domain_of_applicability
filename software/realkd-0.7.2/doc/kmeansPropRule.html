kmeansPropRule

Propositionalization rule for metric attributes that creates a set of propositions corresponding to a binning that is neither equal count nor equal width but instead approximately minimizes sum of squared error.

supertypes

propRule, tobject

subtypes

none

fields

name type description
maxNumberOfIterations posInt Maximal number of kmeans iterations to be performed (default 25)
numberOfCutoffs evenPosInt Even number greater or equal to 2.
