KDON documentation - realKD 0.7.2

standardPropScheme

Propositionalizes table by applying a set of propositionalization rules to each attribute separately.

supertypes

propositionalizationScheme, tobject

subtypes

none

fields

name type description
additionalRules harray(propRule) Additional rules that are applied after the default rules.
defaultCategoricRule propRule Third rule to be applied (default creates equality proposition for all values of categoric attribute.)
defaultMetricRule propRule First rule to be applied (default finds four cut off points by k-means clustering for metric attributes).
defaultOrdinalRule propRule Second rule to be applied (default creates count-based inequality conditions for ordinal attributes that are not metric).
specialRules harray(propRule) Exclusive rules that are applied first to attribute and, when firing, prevent further rules from being applied.
